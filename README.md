# ToDo List Vue App

[![Vue.js](https://img.shields.io/badge/Vue.js-2.7-green.svg)](https://vuejs.org/) [![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

A simple yet elegant **ToDo List application** built with Vue.js 2. Manage your daily tasks effortlessly: add new todos, delete individual or all items, with a beautiful responsive design featuring gradient backgrounds and smooth interactions.

## ✨ Features
- ✅ Add new todo items via input field
- ✅ View numbered list of todos
- ✅ Delete individual todos with 'X' button
- ✅ Clear all todos with dedicated button
- ✅ Pre-populated sample tasks
- ✅ Responsive design (mobile-friendly)
- ✅ Modern UI with Google Fonts (Nunito) and CSS gradients

## 🛠 Tech Stack
| Category | Technologies |
|----------|--------------|
| Framework | Vue.js 2.7 |
| Build Tool | Webpack 3, vue-loader |
| Styling | Sass/SCSS, CSS3 Gradients |
| Fonts | Google Fonts (Nunito) |

## 🚀 Quick Start

### Prerequisites
- Node.js (v8+ recommended)

### Installation
```bash
# Clone the repo
git clone <your-repo-url>
cd "To Do list"

# Install dependencies
npm install
```

### Development
```bash
# Serve with hot reload at http://localhost:8080
npm run dev
```

### Build for Production
```bash
# Build optimized dist for production
npm run build
```
Output in `/dist`.
